Digital Drawing

I have been drawing since I can remember, so I was excited to learn how to draw using procreate on my iPad. With my digital drawings, I really like experimenting with different textures and colors, as well as multiple effects, to try to make the drawing more interesting. All of these drawings were made using procreate on the iPad with an Apple Pencil.
